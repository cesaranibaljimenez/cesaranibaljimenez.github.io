PacMen App

Descripción

Repositorio de GitHub

La app tiene un botón Add PacMan a través del cuál se crean la galletas en la pantalla, cuando se acciona el botón Start Game las galletas creadas se mueven con velocidades y posiciones aleatorias (eje "X" y "Y") hacia los bordes de la pantalla, al chocar con estos las galletas cambian de dirección y se dirigen hacia el borde contrario de la pantalla para repetir el ciclo. Para un mayor detalle de como usar la app y tener acceso a sus archivos de configuración haz click en el reposirtorio de GitHub
